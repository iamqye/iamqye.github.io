Install Hugo

Hugo adalah salah satu generator situs statis open-source yang paling populer. Dengan kecepatan dan fleksibilitasnya yang luar biasa, Hugo membuat membangun situs web menjadi menyenangkan.

Instalasi Hugo pada windows

Jika Anda menggunakan sistem operasi Windows, anda dapat menggunakan Chocolatey untuk manajemen paket, Anda dapat menginstal Hugo dengan satu-baris berikut:

choco install hugo -confirm

Install Chocolatey di Windows
